## About Me

- 🎓 Software engineering background, now focused on **AI-native product engineering** rather than a single backend stack.
- 🧠 I build with **LLMs, agent workflows, embeddings, RAG, computer vision, and full-stack web technologies**.
- 🚀 I enjoy taking a concept through the complete loop: **research → prototype → implementation → evaluation → deployment**.
- 🎯 Current interests: AI for careers and work, multimodal applications, responsible AI, and human–AI collaboration.

> **My view of vibe coding:** use AI to compress the idea-to-product loop, while keeping architecture, testing, observability, explainability, and fallback paths under human control.
